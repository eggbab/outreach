# Outreach — 기능 설명서

B2B 영업 올인원 자동화 플랫폼

---

## 한 줄 요약

**키워드만 입력하면 잠재고객을 자동 수집하고, 이메일·DM을 자동 발송하고, 파이프라인으로 딜을 관리하는 B2B 영업 도구입니다.**

---

## 전체 흐름

```
키워드 입력 → 잠재고객 수집 → 승인/거절 → 이메일·DM 발송 → 반응 추적 → 딜 관리 → 계약 성사
```

---

## 1. 회원가입 & 로그인

- 이메일, 비밀번호, 이름으로 가입합니다.
- 가입 즉시 **Pro 플랜 14일 무료 체험**이 시작됩니다 (카드 등록 없음).
- 체험 기간이 끝나면 자동으로 무료 플랜으로 전환됩니다.

---

## 2. 프로젝트

하나의 영업 캠페인 단위입니다.
예: "서울 식자재 업체 영업", "인테리어 업체 광고 제안"

- 프로젝트를 만들고, 그 안에서 키워드 추가 → 수집 → 발송을 진행합니다.
- 무료 플랜은 1개, Pro/에이전시는 무제한으로 만들 수 있습니다.

---

## 3. 잠재고객 수집

### 이렇게 씁니다
1. 프로젝트에 들어가서 **키워드를 추가**합니다 (예: "업소용 냉장고").
2. 수집할 **소스를 선택**합니다 (네이버, 구글, 인스타그램, 네이버쇼핑, 네이버지도).
3. **"수집 시작"** 버튼을 누르면 자동으로 업체 정보를 긁어옵니다.

### 수집되는 정보
- 업체명, 이메일, 전화번호, 인스타그램 계정, 웹사이트, 업종 카테고리

### 수집 중에 보이는 것
- 진행률 바 (예: "3/10 완료")
- 현재 처리 중인 작업 (예: "네이버에서 '업소용 냉장고' 수집중...")
- 완료 시 총 수집 건수

### 수집 한도 (일일)
| 플랜 | 일일 수집 |
|------|----------|
| 무료 | 50건 |
| 프로 | 500건 |
| 에이전시 | 2,000건 |

---

## 4. 잠재고객 관리

수집된 업체 목록을 검토하고 관리합니다.

### 기본 기능
- **승인/거절**: 수집된 업체를 하나씩 또는 전체 승인. 승인된 업체에만 이메일·DM이 발송됩니다.
- **스코어링**: 시스템이 자동으로 유망도 점수(0~100)를 매깁니다.
  - 이메일 있으면 +10점, 전화번호 있으면 +10점
  - 이메일을 열어봤으면 +15점, 링크를 클릭했으면 +25점
- **태그**: "VIP", "재연락 필요" 같은 태그를 만들어 업체에 붙일 수 있습니다.
- **메모**: 업체별로 메모를 남길 수 있습니다 (예: "담당자 김 부장, 오후 통화 선호").

### 상세 보기
업체를 클릭하면 오른쪽에 패널이 열립니다:
- **정보 탭**: 전화, 인스타, 웹사이트, 스코어 등 상세 정보
- **메모 탭**: 메모 추가/삭제
- **태그 탭**: 태그 붙이기/떼기
- **타임라인 탭**: 이 업체와의 모든 활동 기록 (이메일 발송, 통화, 딜 생성 등)

### CSV 내보내기
수집된 잠재고객 전체를 CSV 파일로 다운로드할 수 있습니다 (한글 Excel 호환).

## 5. 이메일 발송

### 사전 설정 (1회)
설정 페이지에서 Gmail 주소와 **앱 비밀번호**를 입력합니다.
(Google 계정 → 보안 → 2단계 인증 → 앱 비밀번호에서 생성)

### 단일 발송
1. 설정에서 이메일 템플릿을 작성합니다.
2. 프로젝트에서 **"이메일 발송"** 버튼을 누르면, 승인된 업체 중 이메일이 있는 곳에 자동 발송됩니다.
3. 발송 중 진행 상황을 실시간으로 볼 수 있습니다.

### 테스트 발송
본인 이메일로 먼저 테스트 메일을 보내볼 수 있습니다.

### 발송 한도 (일일)
| 플랜 | 일일 이메일 |
|------|-----------|
| 무료 | 10건 |
| 프로 | 100건 |
| 에이전시 | 500건 |

### 템플릿 변수
이메일 본문에 `{company_name}`, `{email}`, `{phone}` 같은 변수를 넣으면 업체별로 자동 치환됩니다.

---

## 6. 이메일 시퀀스 (자동 후속 이메일)

한 번만 설정하면 조건에 따라 후속 이메일이 자동으로 발송됩니다.

### 이렇게 씁니다
1. 시퀀스를 만들고 이름을 정합니다 (예: "초기 영업 시퀀스").
2. **스텝을 추가**합니다:
   - 스텝 1: 즉시 발송 — "안녕하세요, 파트너십을 제안드립니다"
   - 스텝 2: 3일 후, **미열람 시에만** — "혹시 이전 메일 확인하셨나요?"
   - 스텝 3: 5일 후, **미클릭 시에만** — "사례를 공유드립니다"
3. 잠재고객을 시퀀스에 **등록**합니다.
4. 시퀀스를 **활성화**하면 15분마다 자동으로 처리됩니다.

### 발송 조건
- **항상**: 무조건 발송
- **미열람 시**: 이전 이메일을 열어보지 않은 사람에게만 발송
- **미클릭 시**: 이전 이메일의 링크를 클릭하지 않은 사람에게만 발송

### 관리
- 시퀀스를 일시중지/재개할 수 있습니다.
- 등록된 잠재고객의 현재 단계와 다음 발송 예정일을 볼 수 있습니다.

---

## 7. 이메일 템플릿 & A/B 테스트

### 템플릿 라이브러리
자주 쓰는 이메일을 템플릿으로 저장해두고 재사용할 수 있습니다.

### A/B 테스트
하나의 템플릿에 **변형(Variant)**을 만들 수 있습니다:
- 변형 A: "파트너십 제안드립니다" (제목)
- 변형 B: "매출 30% 성장 사례를 공유합니다" (제목)

각 변형의 **발송 수, 열람 수, 클릭 수, 열람률, 클릭률**을 비교해서 어떤 제목이 더 효과적인지 확인할 수 있습니다.

---

## 8. 발송 건강도 체크

이메일을 보내기 전에 **스팸에 걸릴 확률을 점검**합니다.

### 체크 항목
- 스팸 키워드 포함 여부 ("무료", "긴급", "100%" 등)
- 제목이 전체 대문자인지
- 느낌표/물음표 과다 사용
- 본문 길이 (너무 짧거나 너무 긴지)
- 수신거부 안내 포함 여부
- Gmail 설정 완료 여부

### 결과
0~100점 스코어 + 구체적인 문제점 목록 + 개선 제안

---

## 9. 인스타그램 DM 자동 발송

### 작동 방식
1. 크롬 확장프로그램을 설치합니다.
2. 인스타그램에 로그인된 상태에서 확장프로그램이 DM을 자동 발송합니다.
3. 서버에서 DM 대기열을 관리하고, 발송 결과를 기록합니다.

### DM 한도 (일일)
| 플랜 | 일일 DM |
|------|--------|
| 무료 | 5건 |
| 프로 | 30건 |
| 에이전시 | 100건 |

### 확인 가능한 정보
- 확장프로그램 연결 상태 (연결됨/끊김)
- DM 발송 대기열 (아직 안 보낸 업체 목록)
- DM 발송 로그 (성공/실패 기록)

---

## 10. 이메일 트래킹

발송한 이메일의 **열람**과 **링크 클릭**을 실시간으로 추적합니다.

- 이메일에 투명 픽셀을 삽입해서 열람 여부를 감지합니다.
- 본문의 링크를 트래킹 링크로 변환해서 클릭 여부를 감지합니다.
- 대시보드에서 열람 수, 클릭 수를 확인할 수 있습니다.

---

## 11. 분석 (Analytics)

프로젝트별로 영업 성과를 분석합니다.

### 이메일 성과
- 총 발송 수, 열람 수, 클릭 수
- 열람률 (%), 클릭률 (%)

### 일별 통계
- 최근 30일간 일별 발송/열람/클릭 추이 차트

### 전환 퍼널
수집 → 승인 → 이메일 발송 → 열람 → 클릭 각 단계의 전환율을 시각화합니다.

---

## 12. 영업 파이프라인 (CRM)

영업 기회(딜)를 칸반 보드 형태로 관리합니다.

### 기본 스테이지 (커스텀 가능)
리드 → 컨택 → 미팅 → 제안 → 계약 → **성사** / **실패**

### 이렇게 씁니다
1. 잠재고객에서 **"딜 생성"**을 누릅니다.
2. 딜 제목, 금액(원), 프로젝트를 입력합니다.
3. 딜 카드를 **드래그앤드롭**으로 스테이지 사이에 이동합니다.
4. "성사" 또는 "실패" 스테이지로 이동하면 자동으로 종료 처리됩니다.

### 통계
- 전체 딜 수, 총 금액
- 성사 건수, 성사 금액
- 실패 건수

---

## 13. 통화 기록

영업 통화를 기록하고 콜백 스케줄을 관리합니다.

### 기록 항목
- 통화 유형: 발신 / 수신 / 부재중
- 통화 시간 (초 단위)
- 결과 (예: "미팅 확정", "거절", "재통화 요청")
- 메모
- 콜백 일시 (설정하면 콜백 스케줄에 표시)

### 콜백 리마인더
콜백 일시를 설정해두면 페이지 상단에 **"오늘 콜백할 곳"** 목록이 표시됩니다.

---

## 14. 제안서

제안서를 작성하고 발송한 뒤, 고객이 열어봤는지 추적합니다.

### 이렇게 씁니다
1. **"새 제안서"**를 만들고 제목, 내용(HTML), 금액을 입력합니다.
2. **"발송"**을 누르면 고유한 링크가 생성됩니다.
3. 이 링크를 고객에게 이메일이나 메신저로 보냅니다.
4. 고객이 링크를 열면 **자동으로 "열람" 상태**로 변경되고, 열람 시각이 기록됩니다.

### 제안서 상태
초안 → 발송 → 열람 → 수락

---

## 15. 미팅 예약

### 내부 미팅 관리
직접 미팅을 생성하고 관리합니다 (제목, 일시, 시간).

### 가용 시간 설정
요일별 가용 시간을 설정합니다 (예: 월~금 09:00~17:00, 30분 단위).

### 공개 예약 링크
설정한 가용 시간을 기반으로 **공개 예약 페이지**가 생성됩니다.
이 링크를 고객에게 보내면, 고객이 직접 이름/이메일/희망 시간을 입력해서 미팅을 예약합니다.

---

## 16. 활동 타임라인

잠재고객별로 **모든 활동이 자동 기록**됩니다:
- 이메일 발송/열람/클릭
- DM 발송
- 통화 기록
- 딜 생성/스테이지 이동
- 미팅 예약
- 제안서 발송

잠재고객 상세 패널의 "타임라인" 탭에서 시간순으로 확인할 수 있습니다.

---

## 17. 팀 협업

### 이렇게 씁니다
1. **팀을 생성**합니다 (예: "영업 1팀").
2. 이메일 주소로 **멤버를 초대**합니다.
3. 멤버에게 **관리자** 또는 **멤버** 역할을 부여합니다.
4. 프로젝트를 팀에 **공유**합니다.

### 역할
- **관리자**: 멤버 초대/제거, 역할 변경, 프로젝트 공유 가능
- **멤버**: 공유된 프로젝트 접근 가능

---

## 18. API 키

외부 시스템과 연동하기 위한 API 키를 발급합니다.

### 이렇게 씁니다
1. **"새 API 키"**를 클릭하고 이름을 입력합니다 (예: "Zapier 연동").
2. 키가 생성되면 **한 번만 표시**됩니다 — 복사해서 안전하게 보관합니다.
3. 이 키를 `Authorization: Bearer osk_...` 형태로 API 호출에 사용합니다.
4. 더 이상 필요 없으면 **비활성화**할 수 있습니다.

---

## 19. 온보딩 가이드

가입 후 대시보드에 **시작 가이드**가 표시됩니다:

1. 첫 프로젝트 만들기
2. 키워드 추가하기
3. 잠재고객 수집하기
4. Gmail 설정하기
5. 첫 이메일 보내기

각 단계를 완료하면 체크 표시가 되고, 모두 완료하거나 "닫기"를 누르면 사라집니다.
